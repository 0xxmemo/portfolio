# 0xMemo portfolio

Static site built with [Astro](https://astro.build), Tailwind CSS v4, anime.js, and Swiper. Use **Bun** for installs and scripts (this repo is tracked with `bun.lock`).

## Commands

```bash
bun install
bun run dev
bun run build
bun run preview
bun run lint
```

## Requirements

- [Bun](https://bun.sh) 1.1+ (`package.json` lists `engines.bun`).
